Technical Overview of PSES Analysis Python Program

This program overcomes a common shortcoming of survey analysis. Though survey analysis may identify general issues, and even how these issues affect particular groups, it does _not_ identify the bundles of associated questions that would help identify what action management needs to take to correct an issue.

This program overcomes this shortcoming by not only identifying groups that have a below-average survey score, but also by identifying associated questions that also have below-average scores. This identification of associated questions makes more focussed corrective action possible.

This program does _not_ aim to prove the statistical validity of the human-performance-management hypotheses that were used to construct the survey. It assumes these hypotheses are correct, and then undertakes to find instances of the troublesome associations that are predicted by management theory.

Because this program does not depend on conventional statistical methodologies, such as Pearson or Spearman, sample size is not an issue. In addition, rather than requiring raw data, this program makes use of data from the group level and has high tolerance for blanked scores, which means that it can be used in circumstances where confidentiality is important.

A note of caution. For lack of statistical validity, these results are of most value when used as guidelines for cultural auditing and investigation.
